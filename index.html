Ferramentas Utilizadas

Nessa aula foi apresentado o Visual Studio Code a ferrementa que será usada no desenvolvimento, o VSCode, explicou como funciona a area de trabalho do VSCode e ensinou a instalar e utilizar plugins, com o LiveServer que ajuda bastante no desenvolvimento.

Inspetor de Elementos

Nessa aula foi apresentado o Inspetor de Elementos que é aberto ao apertar a tecla "F12" ou "Ctrl+Shift+i", e expliocou o funcionamento da ferramenta.

Estrutura Básica do HTML

Nessa aula foi explicado a estrutura básica do HTML, Explicou que o conteúdo do "body" é o que será exibido e "head" o que é carregado antes de exibir a página.

Falando de Tags

Nessa aula foi explicado o funcionamento das tags, apresentou tags com "i" que transforma o conteúdo entre a tag em italico, entre "strong" transforma em negrito, e finalizou explicou as tags que são fechadas no final da tag antes da chave final.

Atributos das Tags

Nessa aula foi explicado que atributos são propriedades das tags, explicou que existem atributos que todas as tags vão ter como: id, nome, style e class. Eplicou atributos específicos como: number e color, explicou como usar a tag "img" e como manipular os atributos da imagem.

Textos

Nessa aula foi apresentado as tags usadas na tipografia, explicou os h's e o uso do "p" como parágrafo, explicou que o prórpio navegador define a margem automáticamente através das tags mas também pode ser defido por css, explicou como fazer um citação usando a tag "blockquote" ou "q", explicou como sublinhar e marcar o texto usando as tags "u" e "mark" e finalizou explicando a tag "sup" e sua utilização.

Listas Ordenadas e Não Ordenadas

Nessa aula foi explicado as listas ordenadas e não ordenadas "ol" e "ul" respectivamente, explicou que cada item listado deve estar dentro da tag filha "li" para o seu devivo funcionamento e finalizou explicando o a diferença entre a lista ordenada e a lista não ordenada.

Links

Nessa aula foi explicado o funcionamento dos links usando a tag "a" para ancorar a algum endereço, explicou que o endereço vai dentro do recurso "href", explicou o recurso "target" e como funcionam os atributos "_blank" que abre em uma nova aba e o "_self" que abre na própria janela,explicou o recurso "title" que exibe um texto quando o mouse fica sobre o link, e finalizou explicando como fazer a ancoragem dentro do próprio site.
